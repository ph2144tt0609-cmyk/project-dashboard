# project-dashboard

しずく／緑ヶ丘薬局グループの各プロジェクト進捗を1ページで見るダッシュボード。

- データは **AES-256-GCM** で暗号化して `index.html` に埋め込まれており、**合言葉を知っている人だけ**がブラウザ内で復号して閲覧できる。
- リポジトリ・ページが公開でも、合言葉なしでは中身（プロジェクト名・進捗）は復元できない（埋め込まれているのは暗号文のみ）。
- 検索エンジン除けに `noindex` を指定。

合言葉はビルド時の環境変数 `DASH_PASS`（このリポジトリには保存しない）。
